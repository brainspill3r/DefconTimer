# DefconTimer
Defcon Yearly Countdown Timer with Ollama intergration.

Fill in the nesscessary config parts found in > settings.go 

 - defconDate = ""
 - Token = ""
 - config.ChannelID = ""
 - config.BotID = ""

Have Ollama installed.
